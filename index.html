Dorothy es un robot seguidor de línea

Usa diferentes sensores para tomar decisiones autónomas y seguir un camino predefinido

El nombre DOROTHY proviene de

la película El Mago de Oz, donde la protagonista, Dorothy, sigue un camino de baldosas amarillas. De manera similar, nuestro robot sigue una línea, aunque en este caso es blanca.

+ COMPONENTES

Un robot seguidor de línea debe cumplir su mayor misión: seguir la línea. Este objetivo se logra gracias a diferentes componentes ensamblados correctamente, junto con un código que permite al robot tomar decisiones autónomas.

Sensores

Un rastreador detecta la línea a seguir por medio de sensores. Aunque existen muchos tipos, utilizamos sensores infrarrojos (IR) por su costo y practicidad. Dorothy incorpora una barra de 16 sensores para una detección precisa.

Placa de Desarrollo

Elegimos el ESP32 DEVKIT V1 debido a su gran cantidad de pines, conectividad Bluetooth y Wi-Fi. Esta placa nos permite programar y controlar todos los componentes del robot de manera eficiente.

SHIELD

Utilizamos una Placa de Expansión para ESP32 diseñada específicamente para nuestra placa de desarrollo. Esta placa facilita las conexiones entre componentes y mejora la estabilidad del sistema.

Motores

Los motores permiten el movimiento del robot con velocidad controlada. Seleccionamos motores que ofrecen un equilibrio óptimo entre torque y velocidad para asegurar un rendimiento eficiente en distintas superficies.

Motores-reductores

Disminuyen la velocidad de un motor y controlan la fuerza, lo que permite girar o reducir velocidad. Son esenciales para la precisión en los movimientos, especialmente en las curvas y cambios de dirección.

Llantas

Las llantas permiten que el robot se desplace y gire. Seleccionamos llantas livianas con alta adherencia y la suficiente tracción para mantener estabilidad a alta velocidad en diferentes superficies.

Cables

Permiten conectar los diferentes componentes del circuito. Utilizamos cables de alta calidad con la longitud adecuada para asegurar conexiones fiables entre todos los componentes electrónicos.

Rueda loca

Sirve para estabilizar el movimiento del robot y mantenerlo horizontal. También ayuda a que el robot gire con mayor facilidad, complementando el funcionamiento de las ruedas principales.

Chasis

Es la estructura principal que permite el movimiento y contiene el tren motriz. Todos los componentes están montados sobre el chasis, diseñado para proporcionar estabilidad y protección.

+ EVOLUCIÓN DEL PROYECTO

Desde prototipos hasta el modelo final, nuestro robot ha evolucionado a través de varias versiones

V.1

Primer Prototipo

Nuestro primer acercamiento al concepto del robot seguidor de línea, enfocado en probar los sensores básicos.

V.2

Mejoras en Chasis

Rediseño del chasis para mejor estabilidad y optimización de la distribución de componentes.

V.3

Integración de Sensores

Implementación de la barra de 16 sensores para mejorar la precisión del seguimiento de línea.

V.4

Ajustes de Velocidad

Optimización del algoritmo para mejorar la velocidad y reducir las oscilaciones en curvas.

V.5

Prototipo Avanzado

Integración de motores-reductores de alta eficiencia y ajustes finales del chasis.

FINAL

Dorothy

Versión final con todos los componentes optimizados y algoritmos refinados para máximo rendimiento.
